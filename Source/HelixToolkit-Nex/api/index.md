# API Documentation

This section contains the complete API reference for all Helix Toolkit NEX libraries.

## Namespaces

### Core Libraries

- **HelixToolkit.Nex.Graphics** - Core graphics abstraction layer providing platform-independent graphics APIs
- **HelixToolkit.Nex.Graphics.Vulkan** - Vulkan implementation of the graphics layer
- **HelixToolkit.Nex.Maths** - Comprehensive mathematics library for 3D graphics including vectors, matrices, and geometric primitives
- **HelixToolkit.Nex** - Core utilities, logging, memory management, and helper functions

### High-Level Libraries

- **HelixToolkit.Nex.Scene** - Scene graph management and spatial organization
- **HelixToolkit.Nex.Rendering** - High-level rendering components and effects
- **HelixToolkit.Nex.ImGui** - Dear ImGui integration for debug interfaces
